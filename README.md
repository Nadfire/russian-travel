# Проект: Путешествия по России

### Описание проекта

Учебный проект 2 спринта курса Веб-разработчик плюс от Яндекс Практикум.
Проект рассказывает о путешествиях по России.

### Используемые технологии:

- HTML
- CSS
- Grid-Layout
- Flexbox
- BEM Nested
- Адаптивная верстка
- Медиазапросы
